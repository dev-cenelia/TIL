# Scope 정리

## 1. 스코프 종류
- 전역 스코프
- 함수 스코프
- 블록 스코프 (ES6 이후 let/const 도입으로 확립)

## 2. var / let / const 차이
- var: 함수 스코프만 존재 (블록 스코프 없음)
- let/const: 블록 스코프 존재

## 3. 호이스팅
- var, let, const **모두** 호이스팅된다.
- 차이점:
  - var → 선언이 호이스팅 + 초기값은 `undefined`  
  - let/const → 선언은 호이스팅되지만 **TDZ(Temporal Dead Zone)** 때문에 초기화 전 접근 시 ReferenceError 발생

## 4. ES6 이전 vs 이후
- ES6 이전에는 var만 존재 → 블록 스코프, TDZ 개념이 없었음
- var는 선언 전에 접근해도 undefined가 나와 버그 추적 어려움
- 이를 해결하기 위해 ES6에서 let, const, 블록 스코프, TDZ 도입
- 실무에서는 var 사용을 지양하고 let/const 사용 권장

## 5. 스코프 체인
- 안쪽 스코프는 바깥 스코프에 접근할 수 있음
- 바깥 스코프는 안쪽 스코프에 접근할 수 없음
- 변수 참조 시, 안→밖 순서로 스코프 체인을 따라가며 탐색함
